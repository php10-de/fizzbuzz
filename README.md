A program that prints the numbers from 1 to 100
===

This is a trivial project that prints the numbers from 1 to 100.

## Prerequisites
* php v7.1+
* [composer](https://getcomposer.org/download/)

## Test on localhost
```
cd path/to/this-project
composer install
composer test
```
